
# DSMovie

DSMovie é uma aplicação de classificação de vídeos, no qual as pessoas podem enviar notas de 1 a 5 para os mais diversos filmes disponíveis. 

Essa aplicação é parte do processo de aprendizagem do treinamento "Semana Spring / React" do Devsuperior. 
## Atividade
Back - End: Desenvolvimento CRUD por emprego do Spring Tools Suite (Java e Spring Boot);
Nuvem - Heroku.

Front-End: Desenvolvimento da tela de interação usuário e sistema (React);
Nuvem - Netlify.
